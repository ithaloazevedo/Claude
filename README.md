# Claude

Projeto pessoal para centralizar as minhas interações com o Claude, arquivos criados, contextos, agentes e ser meu auxiliar no dia a dia.

## Objetivo

Servir como base de conhecimento e espaço de trabalho integrado ao [Claude Code](https://docs.anthropic.com/en/docs/claude-code), permitindo:

- Armazenar contextos e referências para sessões futuras
- Organizar agentes e automações
- Centralizar arquivos e artefatos gerados nas interações

## Como Usar

1. Abra este repositório no Claude Code
2. O arquivo `CLAUDE.md` será lido automaticamente, fornecendo contexto ao assistente
3. Use as pastas do projeto para organizar seus arquivos e contextos

## Estrutura

```
├── CLAUDE.md          # Contexto automático para o Claude Code
├── README.md          # Esta documentação
```
